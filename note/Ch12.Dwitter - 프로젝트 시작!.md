## 12.7 트윗 Rest APIs - 프로젝트 셋업 및 구조 만들기💡

- API 만들고 프론트 없이 테스트 하기 위해서 포스트맨을 사용해서 응답을 확인한다.

- `>snippets`에서 자바스크립트 선택한 다음 아래처럼 스니펫을 추가해주면 `rr` 명령어로 미리 정의한 코드를 추가할 수 있다.

```json

"Express Callback":{
  "prefix":"rr",
  "body":[
    "(req, res, next) => {$1}"
  ],
  "description":"Express Callback"
}


```
