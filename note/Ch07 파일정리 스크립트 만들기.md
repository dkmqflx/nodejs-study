## 7.1 스크립트 챌린지 소개 (요구 사항 분석) ❓

- 아이폰과 안드로이드로 찍은 사진과 비디오가 있는 상황

- 사진과 동영상을 분리한다

- video 폴더

  - .mp4, .mov 파일

- captured 폴더

  - .png, .aae 파일

- duplicated 폴더

  - .jpg 파일

  - 아이폰에서 사진을 편집하면 IMG_0710 -> IMG_E0710 이런식으로 E라는 키워드가 붙는다

  - 수정한 파일이 있을 때만, 원본 파일을 넣는다

## 7.2 계획 세우고 사용자 입력 처리 💡

- 디버깅을 하시고 싶으시다면! 🐛

- 인자를 전달해야 하는 경우 아래와 같이 .vscode/launch.json 파일을 설정해 보세요: (디버깅 챕터에서 파일 설정하는법 다뤘었죠?)

```json
{
  "program": "${workspaceFolder}/5-project-photo/photo.js", // 여러분 경로대로 바꿔주세요

  "args": ["test"], // 테스트 하고자 하는 폴더 이름

  "restart": true,

  "runtimeExecutable": "node"
}
```
